## 3. 기능

### 3.1. 사용자

> 사용자 관련 정보를 기반으로 로그인, 수정, 찾기, 마이페이지 등의 기능을 사용할 수 있다.

#### 3.1.1 로그인
- 사용자의 이메일과 비밀번호를 입력하면 로그인 할 수 있도록 하는 기능이다.

#### 3.1.2 회원가입
- 사용자의 이메일, 비밀번호, 이름, 연락처, 생년월일을 입력하고 약관의 모두 동의시 회원가입을 할 수 있도록 하는 기능이다.
